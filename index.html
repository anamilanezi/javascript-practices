{ Lógica de programação com JavaScript }

Repositório criado para o desenvolvimento e prática dos meus estudos em lógica de programação utilizando a linguagem JavaScript durante a mentoria Cloud Girls. Espero mantê-lo em constante evolução e aceito qualquer sugestão ou crítica que possam contribuir com esse processo!

Desenhar com o mouse: carinhosamente apelidado de Paint Pastel Princess, esse programa aplica alguns recursos de interação com canvas em resposta a diferentes comportamentos do mouse que permitem o desenho na tela e condicionais para escolha de cores e tamanho do pincel.

Acerte a nuvem: um jogo para acertar a nuvem que é desenhada de forma aleatória na tela. Nesse jogo foram aplicados recursos de interação com canvas, exibindo diferentes mensagens e realizando a troca de cor do desenho quando o usuário acerta o alvo.

Alura: Programas desenvolvidos no curso de Lógica de Programação I e II oferecidos pela platadorma Alura.

Algoritmos: Nessa pasta alguns programas simples serão adicionados como base para consulta de recursos e soluções. As propostas de exercícios foram retiradas do curso "Introdução à Ciência da Computação com Python - Parte 1" oferecidos pela USP na plataforma Coursera, sendo adaptados para JavaScript.

Par ou ímpar: Programa que recebe um número inteiro e informa se o número é par ou ímpar.

Conversor de segundos: Recebe do usuário um número em segundos transforma ele em dias, horas, minutos e segundos.

Retirar o dígito da dezena: Recebe do usuário um número inteiro e retira o dígito da dezena.

Distância entre dois pontos: Recebe quatro números inteiros, que correspondem a dois pares ordenados no plano cartesiano e calcula a distância entre eles.

Calculadora de Bhaskara: Recebe três valores que correspondem aos coeficientes a, b e c de uma equação de segundo grau e calcula o discriminante (delta). Se o delta for maior que zero, a equação terá dois valores reais e distintos. Se o delta for igual a zero, a equação terá somente um valor real ou dois resultados iguais. Se o delta for menor que zero, a equação não possuirá valores reais.
